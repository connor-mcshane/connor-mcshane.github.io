I'm a Senior Data Engineer and Machine Learning Engineer based in London. Originally from Australia, I made the move over here in 2018. 

I actually studied electrical engineering at university, but somehow ended up working in data. Over the last seven years, I've worked on everything from cloud infrastructure to modern data stacks and machine learning solutions. 

My day-to-day usually involves building end-to-end data systems—whether that's wrangling batch and streaming data, setting up CI/CD pipelines, or turning raw data into something actually useful via dashboards and APIs. I spend a lot of time in the Google Cloud ecosystem, too (and picked up seven GCP certifications along the way).

When I'm not building data pipelines, you'll usually find me out running, playing guitar, or drawing.
